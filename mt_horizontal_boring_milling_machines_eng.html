Ceruti, Scharmann, Pama, Innse, and Toshiba provided the encouragement and background in the design and manufacture of TMG horizontal boring and milling machines. Using these classic designs as a starting platform, TMG has adopted modular concepts that facilitate custom designing of these large machines.
TMG's standard line for CNC controlled HBMs includes:

  • Table type - with fixed column and displaceable table
  • "T" type - with longitudinal column movement for the greatest boring accuracy
  • Floor type - with transverse column movement and working feed on both the ram and quill
  • Spindle sizes are offered from 3.5 inch to 8.5 inch (90 mm to 220 mm).

Standard features include:


  • G.E. Fanuc CNC for USA machines (Siemens, Heidenhain, and manual/DRO optional)
  • Parker, Rexroth or Vickers hydraulic elements
  • Main spindle bearing coolant
  • Ball screws
  • Auto lube system
  • Hydraulic clamping on 5 axis
  • Heidenhain linear and rotary encoders
  • Hardened and ground ways (hydrostatic on large machines)
  • Counterways lined with Turcite

Optional feature include:


  • Automatic tool changer
  • Vertical milling head
  • Universal milling head
  • Thru-spindle coolant
  • Detachable facing plate
  • 2-axis CNC head
  • Various rotary tables and floor plates
